树莓派计算板使用说明

1 硬件清单

（1）Compute Module 3+ 32GB 树莓派计算模块

图1 RaspBerry computer module

（2）树莓派CM3/3+ 计算模块扩展板 外设扩展板

图2 扩展板

（3）树莓派摄像头

图3 imx219 相机

2 软件清单

（1）SDFormatter.exe

图4 SDFormatter
下载地址：
http://www.waveshare.net/w/upload/d/d7/Panasonic_SDFormatter.zip

（2）RPiboot_Setup

图5 rpiboot
下载地址：
http://www.waveshare.net/w/upload/f/f3/Rpiboot_setup.zip

（3）镜像
下载地址：
链接：https://pan.baidu.com/s/1_V2FvepdIcp4huIxdP7ANQ
提取码：weat

3 烧录镜像

（1）提前准备好所需要的树莓派镜像（xxx.img文件）
（2）将CM3/CM3+插入IO Board中，将BOOT ENABLE USB SLAVE跳线帽短接EN一端， 拔掉USB SLAVE1/2/3/4 、Select跳线帽。用USB线将IO Board的USB SLAVE 接口连接到PC。

（3）在rpiboot_setup的安装目录下，找到rpiboot.exe文件，以管理员身份运行。

在成功运行之后，你会发现CM3/CM3+在PC中被识别为一个移动盘。
（注意：CM3+第一次使用的时候，可能需要你在PC的磁盘管理中手动做格式化并分配驱动符，才可以在我的电脑中被发现）使用WinDiskImager.exe烧录工具，选择之前下载的树莓派镜像文件，将镜像烧写到EMMC中（即PC识别出来的盘，注意先格式化）
烧写完成之后，将BOOT ENABLE USB SLAVE跳线帽短接DIS一端，并插回USB SLAVE1/2/3/4 Select跳线帽。
接入电源到POWER接口启动即可。
【注意】
在镜像烧写过程中，注意PC尽量不要对其他USB存储设备进行写操作，避免出现冲突。同时不要触碰usb线缆以及板子，可能会造成烧写中断。

4 连接相机

使用相机接口（cam0 cam1）需要短接以下引脚：

CAM0相关的引脚：

GPIO28 <-> CD0_SDA
GPIO29 <-> CD0_SCL
GPIO30 <-> CAM0_IO1
GPIO31 <-> CAM0_IO0

CAM1相关的引脚：

GPIO0 <-> CD1_SDA
GPIO1 <-> CD1_SCL
GPIO4 <-> CAM1_IO1
GPIO5 <-> CAM1_IO0

Markdown的用法

2020-04-07

Markdown 怎么用

Markdown的语法

1.标题

语法：

# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

2.字体

语法：

**这是加粗的文字**
*这是倾斜的文字*`
***这是斜体加粗的文字***
~~这是加删除线的文字~~

效果：

这是加粗的文字

这是倾斜的文字`

这是斜体加粗的文字

这是加删除线的文字

3.引用

在引用的文字前加>即可

引用也可以嵌套，如加两个>>三个>>>

语法：

>这是引用的内容
>>这是引用的内容
>>>>>>>>>>这是引用的内容

这是引用的内容

这是引用的内容

这是引用的内容

4.分割线

语法：

-----
*****

5.超链接

语法：

[超链接名](超链接地址 "超链接title")
其中 title可加可不加

码云

百度

6.列表

无序列表：
无序列表用 - + * 任何一种都可以

语法：

- 列表内容A
+ 列表内容B
* 列表内容C
  • 列表内容A
  • 列表内容B
  • 列表内容C

注意：- + * 跟内容之间都要有一个空格

有序列表：

1. 列表内容
2. 列表内容
3. 列表内容
  1. 列表内容
  2. 列表内容
  3. 列表内容

列表嵌套

上一级和下一级之间敲三个空格即可

7.图片

语法：

![图片alt](图片地址 ''图片title'')
图片alt就是显示在图片下面的文字，相当于对图片内容的解释。
图片title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加

示例：

![blockchain](https://ss0.bdstatic.com/70cFvHSh_Q1YnxGkpoWK1HF6hhy/it/
u=702257389,1274025419&fm=27&gp=0.jpg “区块链”)

8.表格

语法：

表头|表头|表头
---|:--:|---:
内容|内容|内容
内容|内容|内容

第二行分割表头和内容。

  • 有一个就行，为了对齐，多加了几个
    文字默认居左
  • 两边加：表示文字居中
  • 右边加：表示文字居右

注：原生的语法两边都要用 | 包起来。此处省略

示例：

表头 表头 表头
内容 内容 内容
内容 内容 内容

9.代码

语法：

单行代码：代码之间分别用一个反引号包起来

语法：

`print（“hello world”）`

示例：

print（“hello world”）
